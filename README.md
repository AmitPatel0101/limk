# About

Simple link shortner by annie. for basic next.js(react), mongoDB.js project.
- [Link](https://limk.site)

# Get Started

```bash
    git clone https://github.com/snehilgamit/linkshortner.git

    cd linkshortner

    npm install

    touch .env.local

    # add MongoURI = your mongoDb database uri

    npm run dev
```
